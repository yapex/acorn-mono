## 一、财务指标清单

### 1.1 基础指标（直接从字段计算）

| 指标 | 字段 | 计算公式 |
|------|------|---------|
| 毛利 | revenue, cost | revenue - cost |
| 毛利率 | revenue, cost | (revenue - cost) / revenue |
| 净利率 | net_profit, revenue | net_profit / revenue |
| ROE | net_profit, equity | net_profit / equity |
| EBITDA | operating_profit, depreciation, amortization | operating_profit + depreciation + amortization |
| 自由现金流 | operating_cash_flow, capex | operating_cash_flow - capex |
| 现金净额 | cash, interest_bearing_debt | cash - interest_bearing_debt |

### 1.2 偿债能力指标

| 指标 | 字段 | 计算公式 | 危险阈值 |
|------|------|---------|---------|
| 现金比率 | cash, current_liabilities | cash / current_liabilities | <0.5 🔴 |
| 速动比率 | cash, accounts_receivable, current_liabilities | (cash + accounts_receivable) / current_liabilities | <1 🔴 |
| 流动比率 | current_assets, current_liabilities | current_assets / current_liabilities | <1 🔴 |
| 资产负债率 | total_liabilities, total_assets | total_liabilities / total_assets | >70% 🔴 |
| 净负债率 | interest_bearing_debt, cash, equity | (interest_bearing_debt - cash) / equity | >100% 🔴 |
| 利息保障倍数 | operating_profit, interest_expense | operating_profit / interest_expense | <3 🔴 |
| 经营现金流/债务 | operating_cash_flow, total_debt | operating_cash_flow / total_debt | <10% 🔴 |
| 债务/EBITDA | total_debt, ebitda | total_debt / ebitda | >4 🔴 |

### 1.3 造假识别指标（C01-C10）

| 编号 | 指标 | 计算公式 | 触发阈值 |
|------|------|---------|---------|

| 优先级 | 指标 | 计算公式 | 触发条件 |

### 1.5 长期价值指标

| 指标 | 计算公式 | 评分标准 |
|------|---------|---------|
| 营收 CAGR | (终值/初值)^(1/n) - 1 | 优秀>15% |
| 净利润 CAGR | (终值/初值)^(1/n) - 1 | 优秀>15% |
| 增长一致性 | 正增长年数 / 总年数 | 优秀>80% |
| ROE 波动率 | 标准差 / 均值 | 优秀<20% |
| 毛利率波动率 | 标准差 | 优秀<5% |
| 主营业务占比 | 主营收入 / 总收入 | 优秀>90% |

## 二、CLI 演化流程

### 2.1 现有命令

```bash
# 查询股票数据
acorn vi query 600519 --fields roe,net_profit --years 5

# 列出可用字段
acorn vi list-fields

# 列出可用计算器
acorn vi list-calculators
```

### 2.2 演化触发

当请求的计算器不存在时，系统自动输出演化规范：

```bash
acorn vi query 600519 --calculators gross_profit_margin

# 输出：
# EVOLUTION_NEEDED: calculator/gross_profit_margin
# 
# 要创建计算器，请按以下格式提供代码：
# 
# ```python
# REQUIRED_FIELDS = ["revenue", "cost"]
# 
# def calculate(data, config):
#     return (data["revenue"] - data["cost"]) / data["revenue"]
# ```
```

### 2.3 创建计算器

在 `calculators/` 目录下创建文件：

```python
# calculators/calc_gross_profit_margin.py
"""
毛利/营收
"""
REQUIRED_FIELDS = ["revenue", "cost"]

def calculate(data, config):
    """
    计算毛利率 = (营收 - 成本) / 营收
    """
    return (data["revenue"] - data["cost"]) / data["revenue"]
```

### 2.4 命名规范

- 文件名：`calc_<英文描述>.py`
- 使用完整英文单词表达财务意义
- 不要过长，控制在 3-5 个单词
- 标准缩写可接受（ROE、PE）

**示例：**

| ✅ 正确 | ❌ 错误 | 原因 |
|--------|--------|------|
| `calc_gross_profit` | `calc_gp` | 不要缩写 |
| `calc_net_profit_margin` | `calc_npm` | 不要缩写 |
| `calc_interest_coverage_ratio` | `calc_icr` | 可接受 |
| `calc_return_on_equity` | `calc_roe` | ROE 是标准缩写 |

## 四、实现清单

### 4.1 优先级排序

| 优先级 | 指标 | 理由 |
|--------|------|------|
| P0 | 毛利、净利率、ROE | 最基础，必备 |
| P1 | 资产负债率、利息保障倍数 | 偿债能力核心 |
| P1 | 经营现金流/净利润 | 造假识别核心 |
| P2 | 现金比率、流动比率 | 短期偿债 |
| P2 | CAGR、波动率 | 长期价值 |

### 4.2 目标

通过 CLI 演化，逐步实现上述指标的计算器。
