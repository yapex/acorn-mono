# 第七段：特殊项目分析

分析非常规项目对财务健康的影响，包括审计意见、股东回报、资本开支、减值拨备等。

---

## 1. 需要的数据与计算

### 数据来源

```bash
# 注意：审计意见需手动查阅，acorn vi query 不提供
acorn vi query {代码} --items dividend,share_repurchase,net_profit,capital_expenditure,total_revenue --years 10
```

### 需要的字段

| 类别 | 字段 | 说明 |
|------|------|------|
| 审计 | ❌ 不提供 | 需手动查阅年报 |
| 股东回报 | `dividend`, `share_repurchase`, `net_profit` | 分红和回购 |
| 资本开支 | `capital_expenditure`, `total_revenue` | 投资规模 |
| 减值 | `goodwill`（有），减值数据 ❌ 不提供 | 需手动查阅 |

### 计算公式

| 指标 | 公式 |
|------|------|
| 股东回报率 | `(dividend + share_repurchase) / net_profit` |
| 资本开支占比 | `capital_expenditure / total_revenue` |
| 分红率 | `dividend / net_profit` |

## 3. 核查结果

### 快速检查清单

| 检查项 | 标准 | 结果 | 判定 |
|--------|------|------|:----:|
| 审计意见 | 🟡需手动查阅 | — |
| CFO更换 | 5年≤3次 | 2次 | ✅ |
| 关联交易 | <30%营收 | 25% | ✅ |
| 主营业务变更 | ≤2次 | 1次 | ✅ |
| 重大诉讼 | 无重大不利 | 无 | ✅ |
| 资产抵押 | <20%资产 | 10% | ✅ |

### 风险项

| 检查项 | 结果 | 风险 |
|--------|------|:----:|
| 审计意见带强调事项 | 2024年 | ⚠️ |
| 资本开支占比上升 | 5%→10% | ⚠️ |
| 商誉减值 | 2022年0.5亿 | ⚠️ |

### 综合判定

```
特殊项目：⚠️ 关注
- 审计意见：🟡 需手动查阅年报（acorn vi query 不提供）
- 股东回报：✅ 稳定，回报率30-40%
- 资本开支：⚠️ 占比上升，需关注投向
- 减值拨备：⚠️ 2022年商誉减值
```
