# 第十段：局限性说明

说明本报告的分析局限，确保使用者正确理解报告结论。

---

## 1. 需要的数据

### 数据来源

| 来源 | 内容 |
|------|------|
| 第一段 | 数据年限、完整性 |
| 第三段 | 数据缺失标记（🟡） |
| 综合 | 行业特点 |

---

## 2. 展示样例

### 数据相关局限

| 局限项 | 说明 | 影响 |
|--------|------|------|
| 数据年限 | 10年（2015-2024） | 无法评估更长周期 |
| 数据来源 | 公开财报 | 可能存在滞后 |
| 数据缺失 | gross_profit 2016-2018 | 毛利率分析受限 |
| 货币单位 | 人民币 | A股适用，港美股需换算 |

### 分析框架局限

| 局限项 | 说明 | 影响 |
|--------|------|------|
| 定量为主 | 本框架以定量分析为主 | 定性因素（如管理层能力）难以量化 |
| 历史数据 | 基于历史财务数据 | 不做未来预测 |
| 行业差异 | 阈值基于通用标准 | 金融/地产等特殊行业需调整 |
| 造假检测 | 基于财报比率分析 | 难以检测虚构业务等复杂造假 |

### 行业特殊性说明

| 行业 | 需调整的指标 | 说明 |
|------|-------------|------|
| 金融 | 资产负债率、流动比率 | 金融行业负债天然较高 |
| 地产 | 现金流/净利润 | 期房销售模式导致现金流滞后 |
| 互联网 | 轻资产、用户数据 | 传统财务指标不适用 |

---

## 3. 核查结果

### 框架适用性

| 公司类型 | 适用性 | 说明 |
|----------|:------:|------|
| 制造业 | ✅ 完全适用 | 指标体系匹配 |
| 零售业 | ✅ 基本适用 | 需关注存货/应收 |
| 金融业 | ⚠️ 部分适用 | 需调整阈值 |
| 地产 | ⚠️ 部分适用 | 现金流分析受限 |
| 互联网 | ⚠️ 部分适用 | 轻资产模式需调整 |

### 数据局限性清单

```
本报告局限性：
1. 数据仅覆盖最近10年，无法评估更长周期表现
2. 部分年份数据缺失（gross_profit 2016-2018），分析受限
3. 基于历史财务数据，不对未来业绩做预测
4. 定量分析为主，定性因素（如管理层、行业地位）难以量化
5. 难以检测虚构业务等复杂财务造假
6. 阈值基于通用标准，特殊行业需调整
```
