# 第二段：偿债能力分析（排险）

评估企业偿还债务的能力，分为短期、长期、现金流覆盖三个维度。

---

## 1. 需要的数据与计算

### 数据来源

```bash
acorn vi query {代码} --items cash_and_equivalents,current_assets,current_liabilities,total_assets,total_liabilities,total_equity,interest_bearing_debt,operating_cash_flow,operating_profit,interest_expense,capital_expenditure --years 10
```

### 需要的字段

| 类别 | 字段 |
|------|------|
| 短期偿债 | `cash_and_equivalents`, `current_assets`, `current_liabilities` |
| 长期偿债 | `total_assets`, `total_liabilities`, `total_equity`, `interest_bearing_debt` |
| 现金流 | `operating_cash_flow`, `operating_profit`, `interest_expense`, `capital_expenditure` |

### 计算公式

#### 短期偿债能力

| 指标 | 公式 | 说明 |
|------|------|------|
| 现金比率 | `cash_and_equivalents / current_liabilities` | 最严格的流动性指标 |
| 速动比率 | `(cash_and_equivalents + accounts_receivable) / current_liabilities` | 剔除存货 |
| 流动比率 | `current_assets / current_liabilities` | 传统指标 |

> 注意：港股有 `quick_ratio` 字段可用，A 股需手动计算

#### 长期偿债能力

| 指标 | 公式 | 说明 |
|------|------|------|
| 资产负债率 | `total_liabilities / total_assets` | 负债占比 |
| 净负债率 | `(interest_bearing_debt - cash_and_equivalents) / total_equity` | 有息负债减现金/净资产 |
| 利息保障倍数 | `operating_profit / interest_expense` | 利润覆盖利息能力 |

#### 现金流覆盖

| 指标 | 公式 | 说明 |
|------|------|------|
| 经营现金流/债务 | `operating_cash_flow / total_liabilities` | 债务偿还能力 |
| 自由现金流 | `operating_cash_flow - capital_expenditure` | 可支配现金 |
| 债务/EBITDA | `total_liabilities / (operating_profit + depreciation)` | 偿债周期估算 |

## 3. 核查结果

### 阈值标准

| 指标 | 安全 | 警惕 | 危险 |
|------|:----:|:----:|:----:|
| 现金比率 | >1 | 0.5-1 | <0.5 |
| 速动比率 | >1.5 | 1-1.5 | <1 |
| 流动比率 | >2 | 1-2 | <1 |
| 资产负债率 | <50% | 50-70% | >70% |
| 净负债率 | <50% | 50-100% | >100% |
| 利息保障倍数 | >5 | 3-5 | <3 |
| 经营现金流/债务 | >20% | 10-20% | <10% |

### 异常标记

| 检查项 | 结果 |
|--------|------|
| 现金比率 < 0.5 | 🔴 2020, 2021, 2022（连续3年） |
| 资产负债率 > 70% | ❌ 无 |
| 利息保障倍数 < 3 | 🔴 2023年 |
| 自由现金流连续2年负 | 🔴 2022, 2023 |

### 综合判定

```
偿债能力：⚠️ 警惕
- 短期：⚠️ 警惕（现金比率下降）
- 长期：✅ 安全
- 现金流：🔴 危险（自由现金流连续为负）
```

---

## 4. 判定标准汇总

### 维度否决规则

| 风险等级 | 条件 | 结果 |
|:--------:|------|------|
| 🔴 直接排除 | 现金比率 < 0.5 连续3年 | 偿债能力严重不足 |
| 🔴 直接排除 | 资产负债率 > 70% | 负债过高 |
| 🔴 直接排除 | 利息保障倍数 < 3 | 利润无法覆盖利息 |
| ⚠️ 警告 | 任一指标处于警惕区间 | 需关注 |
