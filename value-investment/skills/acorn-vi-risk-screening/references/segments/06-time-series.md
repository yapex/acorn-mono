# 第六段：时间序列分析

逐年追踪核心指标，识别持续恶化的趋势信号。

---

## 1. 需要的数据与计算

### 数据来源

```bash
acorn vi query {代码} --items total_revenue,net_profit,gross_profit,roe,operating_cash_flow,inventory,accounts_receivable --years 10
```

### 需要的字段

| 类别 | 字段 | 说明 |
|------|------|------|
| 增长 | `total_revenue`, `net_profit` | 计算同比增速 |
| 盈利 | `gross_profit`, `roe` | 毛利率、ROE |
| 现金流 | `operating_cash_flow` | 与净利润对比 |
| 资产 | `inventory`, `accounts_receivable` | 增速对比 |

### 计算公式

```python
# 同比增速
revenue_growth = (revenue_t - revenue_t-1) / revenue_t-1
profit_growth = (profit_t - profit_t-1) / profit_t-1

# 现金流/净利润
cfo_to_profit = operating_cash_flow / net_profit

# 毛利率
gross_margin = (revenue - gross_profit) / revenue

# 存货/应收增速
inventory_growth = (inventory_t - inventory_t-1) / inventory_t-1
receivable_growth = (receivable_t - receivable_t-1) / receivable_t-1
```

## 3. 核查结果

### 趋势判断

| 指标 | 10年趋势 | 判定 |
|------|----------|:----:|
| 营收增速 | ↓ 15% → 10% | ⚠️ 下降 |
| 净利润增速 | ↓ 18% → 12% | ⚠️ 下降 |
| 毛利率 | → 40-45% | ✅ 稳定 |
| ROE | → 10-15% | ✅ 稳定 |
| 现金流/净利润 | ↓ 1.2 → 0.6 | 🔴 恶化 |

### 关键转折点

| 年份 | 事件 | 影响 |
|------|------|------|
| 2019 | 营收增速首次跌破10% | ⚠️ |
| 2020 | 疫情冲击，营收下降5% | ⚠️ 疫情影响 |
| 2021 | 疫后复苏，但现金流恶化 | 🔴 |
| 2022-2023 | 现金流持续低于净利润 | 🔴 |

### 综合判定

```
时间序列分析：🔴 红灯
- 现金流/净利润：6年低于1
- 存货/营收：持续上升至42%
- 应收/营收：持续上升至40%
- 建议：深入调查现金流质量问题
```
