# 第三段：组合信号排查（排雷）

关注"财务比率的异常组合"，捕捉造假模式。与第四段**互补**：
- 第三段：看比率背离（收入↑ 但 现金流↓）
- 第四段：看绝对值超标（应收 > 营收30%）

---

## 1. 需要的数据与计算

### 数据来源

```bash
acorn vi query {代码} --items operating_cash_flow,net_profit,inventory,total_revenue,accounts_receivable,cash_and_equivalents,total_assets,construction_in_progress,other_receivables,related_party_transactions,goodwill,depreciation_rate --years 10
```

### 需要的字段

| 信号 | 字段 | 说明 |
|------|------|------|
| C01 | `operating_cash_flow`, `net_profit` | 累计10年 |
| C02 | `inventory`, `total_revenue` | 需要增速计算 |
| C03 | `accounts_receivable`, `total_revenue` | 需要增速计算 |
| C03补充 | 注意：A 股（Tushare）应收数据可能仅覆盖近 3-5 年，2016 年前通常缺失 |
| C04 | `interest_income`, `cash_and_equivalents` | 利息收入/货币资金（✅ A股有） |
| C05 | `prepaid_expenses`, `total_assets` | 预付款 |
| C06 | `construction_in_progress`, `total_assets` | 在建工程 |
| C07 | `other_receivables`, `total_assets` | 其他应收 |
| C08 | `related_party_transactions`, `total_revenue` | 关联交易（**A 股不提供，需手动查阅年报**） |
| C09 | `goodwill`, `total_equity` | 商誉（✅ A股有 goodwill） |
| C09 | `goodwill`, `total_equity` | 商誉 |
| C10 | `depreciation_rate` | 折旧政策 |

> **⚠️ 数据缺失处理**：A 股数据源字段限制：
> - `accounts_receivable`：通常仅覆盖 2022-2024 年，2016-2021 年可能为 N/A
> - `audit_opinion`、`related_party_transactions`、`depreciation_rate`：**A 股原始字段不可用**，需手动查阅年报
> - `goodwill` ✅、`interest_income` ✅、`other_receivables` ✅、`prepayment` ✅：A 股有
>
> **处理原则**：数据缺失标记 🟡，不得主观填补。缺失不影响其他指标判定。

### 计算公式

| 信号 | 计算方法 |
|------|----------|
| C01 | 累计10年 `operating_cash_flow / net_profit` |
| C02 | 10年平均 `(Δinventory / inventory) - (Δrevenue / revenue)` |
| C03 | 10年平均 `(Δreceivable / receivable) - (Δrevenue / revenue)` |
| C04 | 10年平均 `interest_income / cash_and_equivalents` |
| C05-C09 | 最新年份 `xxx / total_assets` 或 `xxx / total_equity` |
| C10 | 对比折旧年限变化（定性） |

### 3.4 存货陷阱专项（🪤）

**核心原理**：存货造假（虚增存货）会同时导致三个异常信号，组合出现时风险极高：

| 信号 | 来源 | 判定条件 |
|------|------|----------|
| 存货周转率下降 | C02 分析 | 连续2年下降，或最新值低于5年均值20%以上 |
| 毛利率突然上升 | P1 分析 | 连续2年上升，或突变+10%以上 |
| 经营现金流为负 | C01 分析 | 最近1年 < 0 |

> 💡 **组合效应**：单一信号可能是行业周期变化，但三信号同时出现 = 🔴 存货造假高危

**综合风险等级**：

| 触发组合 | 风险等级 | 说明 |
|----------|:--------:|------|
| 三信号同时 | 🔴 **高危** | 存货造假嫌疑极大，强烈建议回避 |
| 周转率↓ + CFO<0 | 🟠 **中危** | 存货积压且现金流出，需警惕 |
| 毛利率↑ + CFO<0 | 🟠 **中危** | 利润虚高但无现金支撑，需警惕 |
| 周转率↓ + 毛利率↑ | 🟡 **关注** | 经典存货造假组合，但现金流尚可 |
| 单一信号 | ⚪ **正常** | 可能是行业周期或经营策略变化 |

**核查结果样例**：

| 信号 | 状态 | 详情 |
|------|------|------|
| 存货周转率下降 | ⚠️ | 2022-2023 连续下降，低于均值15% |
| 毛利率突然上升 | ✅ | 无突变 |
| 经营现金流为负 | ✅ | >0 |

**综合判定**：

| 维度 | 结果 |
|------|:----:|
| 触发信号数 | 1/3 |
| 风险等级 | ⚪ 正常 |

**存货陷阱风险：⚪ 正常（无三信号组合）**

> ⚠️ **行业差异**：部分行业（如房地产、白酒）存货周转天然较低，需结合行业背景判断。
