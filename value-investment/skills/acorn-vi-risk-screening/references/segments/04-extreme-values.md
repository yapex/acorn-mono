# 第四段：单字段极值排查（排雷）

关注"单项指标的极端值"，捕捉经营失控或造假直接证据。

与第三段**互补不重复**：
- 第三段：看比率背离（收入↑ 但 现金流↓）
- 第四段：看绝对值超标（应收 > 营收30%）

---

## 1. 需要的数据与计算

### 数据来源

```bash
acorn vi query {代码} --items net_profit,operating_cash_flow,inventory,accounts_receivable,total_revenue,total_assets,gross_profit,goodwill,construction_in_progress,depreciation,capital_expenditure,operating_expense,long_term_deferred_expenses,roe --years 10
```

### 需要的字段

| 级别 | 字段 | 说明 |
|------|------|------|
| P0 | `operating_cash_flow`, `net_profit`, `accounts_receivable`, `inventory`, `gross_profit`, `audit_opinion` | 直接排除信号 |

> **⚠️ 数据缺失处理**：A 股数据源限制：
> - `accounts_receivable`：通常仅覆盖 2022-2024 年，判断时注意
> - `audit_opinion`：**A 股不可用**，需手动查阅年报
> - `goodwill`、`long_term_deferred_expenses`：A 股有 ✅
| P1 | `gross_profit`, `goodwill`, `construction_in_progress`, `depreciation`, `operating_expense`, `total_revenue`, `long_term_deferred_expenses` | 异常警示 |
| P2 | `roe`, `total_revenue`, `interest_expense`, `accounts_receivable`, `inventory`, `capital_expenditure`, `depreciation` | 经营警示 |

### 计算公式

| 指标 | 公式 |
|------|------|
| 现金流/净利润 | `operating_cash_flow / net_profit` |
| 应收/营收 | `accounts_receivable / total_revenue` |
| 应收/总资产 | `accounts_receivable / total_assets` |
| 存货/营收 | `inventory / total_revenue` |
| 存货/总资产 | `inventory / total_assets` |
| 毛利率变化 | `gross_profit_t / gross_profit_t-1 - 1` |
| 商誉/净资产 | `goodwill / total_equity` |
| 费用增速 - 营收增速 | `(Δoperating_expense / operating_expense) - (Δrevenue / revenue)` |
| 长期待摊/总资产 | `long_term_deferred_expenses / total_assets` |
| **资本开支/折旧比率** | `capital_expenditure / depreciation` |

## 2. 展示样例

### P0 最高警惕（直接排除信号）

| 指标 | 判定条件 | 最新值 | 年份 | 判定 | 风险 |
|------|----------|--------|------|:----:|:----:|
| 现金流/净利润 | 连续2年 < 1 | 0.8, 0.6 | 2022, 2023 | 🔴 | 🔴 |
| 应收账款 | >营收30% | 35% | 2023 | 🔴 | 🔴 |
| 存货 | >营收50% | 40% | 2023 | ⚠️ | ⚠️ |
| 毛利率 | 突然大幅提升+10% | +12% | 2021 | 🔴 | 🔴 |
| 审计意见 | 非标准无保留 | 🟡需手动查阅 | — | 🟡 | 需手动查阅年报 |

### P1 重点关注（异常警示）

| 指标 | 判定条件 | 最新值 | 年份 | 判定 | 风险 |
|------|----------|--------|------|:----:|:----:|
| 毛利率 | >同行20% | +15% | 2023 | ⚠️ | ⚠️ |
| 毛利率突变 | 连续2年上升 或 单年+10% | +12% | 2021 | ⚠️ | ⚠️ ← **用于存货陷阱** |
| 商誉 | >30%净资产 | 22% | 2023 | ✅ | — |
| 在建工程进度 | <50%持续2年 | 进度65% | — | ✅ | — |
| 折旧政策 | 无理由变更 | 无变化 | — | ✅ | — |
| 费用增速 | >营收增速20% | +18% | 2023 | ⚠️ | ⚠️ |
| 长期待摊 | >5%总资产 | 3% | 2023 | ✅ | — |

### P2 常规关注（经营警示）

| 指标 | 判定条件 | 连续年数 | 判定 | 风险 |
|------|----------|:--------:|:----:|:----:|
| ROE | 连续3年 < 10% | 0年 | ✅ | — |
| 营收增速 | 连续3年 < 5% | 0年 | ✅ | — |
| 利息保障倍数 | < 3 | 0年 | ✅ | — |
| 应收周转率 | 连续2年下降 > 20% | 1年 | ⚠️ | ⚠️ |
| 存货周转率 | 连续2年下降 或 <5年均值20% | 0年 | ✅ | — ← **用于存货陷阱** |
| **资本开支/折旧** | **> 2（过度投资信号）** | 0年 | ✅ | — |

> 💡 **资本开支/折旧比率**：反映公司投资强度。比率 > 2 表示投资支出远超折旧补偿，可能是过度扩张或资产虚高的信号。

### 综合判定

| 级别 | 触发数 | 风险 |
|------|:------:|:----:|
| P0 | 2/5 | 🔴 |
| P1 | 2/6 | ⚠️ |
| P2 | 1/6 | ⚠️ |

## 3. 核查结果

### P0 阈值（直接排除）

| 指标 | 阈值 | 触发即排除 |
|------|------|:----------:|
| 现金流/净利润 | 连续2年 < 1 | 🔴 |
| 应收账款 | >营收30% 或 >总资产20% | 🔴 |
| 存货 | >营收50% 或 >总资产30% | 🔴 |
| 毛利率 | 突然大幅提升（+10%以上） | 🔴 |
| 审计意见 | 非标准无保留 | 🔴（需手动查阅，A 股数据源不可获取） |

### P1 阈值（重点关注）

| 指标 | 阈值 |
|------|------|
| 毛利率 | >同行20% |
| 商誉 | >30%净资产 |
| 在建工程进度 | <50%持续2年 |
| 折旧政策 | 无理由变更 |
| 费用增速 | >营收增速20% |
| 长期待摊费用 | >5%总资产 |

### P2 阈值（常规关注）

| 指标 | 阈值 |
|------|------|
| ROE | 连续3年 < 10% |
| 营收增速 | 连续3年 < 5% |
| 利息保障倍数 | < 3 |
| 应收周转率 | 连续2年下降 > 20% |
| 存货周转率 | 连续2年下降 **或** <5年均值20%（**用于存货陷阱**） |
| **资本开支/折旧** | **> 2（过度投资信号）** |

### 风险等级

| 触发情况 | 风险等级 |
|----------|:--------:|
| P0 触发任一项 | 🔴 直接排除 |
| P1 触发 ≥3 项 | ⚠️ 重点关注 |
| P2 触发 ≥3 项 | ⚠️ 经营警示 |
| 全部正常 | ✅ 通过 |
