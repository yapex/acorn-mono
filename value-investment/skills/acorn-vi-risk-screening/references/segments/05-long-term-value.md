# 第五段：长期价值分析（排优）

评估企业作为长期投资标的的可行性，从增长性、稳定性、可理解性、穿越周期四个维度评分。

---

## 1. 需要的数据与计算

### 数据来源

```bash
acorn vi query {代码} --items total_revenue,net_profit,roe,gross_profit,operating_cash_flow --years 10
```

### 需要的字段

| 维度 | 字段 | 说明 |
|------|------|------|
| 长期增长 | `total_revenue`, `net_profit` | 计算10年CAGR |
| 稳定性 | `roe`, `gross_profit`, `operating_cash_flow` | 计算波动率 |
| 可理解性 | `total_revenue` | 主营占比（需定性分析） |

### 计算公式

#### 长期增长（权重25%）

```python
# CAGR = (终值/初值)^(1/年数) - 1
revenue_cagr = (revenue_latest / revenue_earliest) ** (1/9) - 1
profit_cagr = (profit_latest / profit_earliest) ** (1/9) - 1
growth_score = (revenue_cagr + profit_cagr) / 2
```

#### 稳定性（权重35%）

```python
# 波动率 = 标准差/均值
roe_volatility = roe.std() / roe.mean()
margin_volatility = margin.std() / margin.mean()
cfo_volatility = cfo.std() / cfo.mean()

# 稳定性得分：波动率越低，分数越高
stability_score = 100 - (roe_volatility + margin_volatility + cfo_volatility) / 3 * 100
```

#### 可理解性（权重20%）

需定性判断：
- 主营占比 > 80% → 20分
- 主营占比 60-80% → 15分
- 主营占比 < 60% → 5分

#### 穿越周期（权重20%）

| 情况 | 得分 |
|------|:----:|
| 历次危机中 ROE 保持 > 10% | 20分 |
| 危机中 ROE 下降但未亏损 | 15分 |
| 危机中 ROE 多次为负或 < 5% | 5分 |
