# 第一段：核心数据展示

财务快照，让读者快速了解公司全貌。

---

## 1. 需要的数据与计算

### 数据来源

```bash
acorn vi query {代码} --items net_profit,total_revenue,total_assets,total_equity,operating_cash_flow,gross_profit,operating_profit,total_liabilities,cash_and_equivalents,interest_bearing_debt,roe --years 10
```

### 需要的字段

| 类别 | 字段 |
|------|------|
| 利润表 | `total_revenue`, `net_profit`, `gross_profit`, `operating_profit` |
| 资产负债表 | `total_assets`, `total_equity`, `total_liabilities` |
| 现金流量表 | `operating_cash_flow` |
| 计算器 | `roe` |

### 计算公式（内部计算）

| 指标 | 公式 |
|------|------|
| 毛利率 | `(total_revenue - gross_profit) / total_revenue` |
| 净利率 | `net_profit / total_revenue` |
| 资产负债率 | `total_liabilities / total_assets` |
| 现金流/净利润 | `operating_cash_flow / net_profit` |
| 净资产收益率 | 直接取 `roe` 字段 |
