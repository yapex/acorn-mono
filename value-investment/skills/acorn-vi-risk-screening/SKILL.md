# 企业财务排雷系统 🔍

三位一体分析框架：**排雷（造假）→ 排险（生存）→ 排优（长期价值）**，共十段分析流程、54 项自检。

> **定位**：排雷工具，不是选股工具。排除有问题的公司，不保证没问题的公司就是好公司。
> **原则**：定量为主（≥90%），定性为辅；基于历史数据，不做预测。

## 第一步：数据获取

### 一条命令获取所有数据

```bash
acorn vi query {股票代码} \
  --items gross_margin,net_profit_margin,roe,ebitda,cash_ratio,current_ratio,quick_ratio,debt_ratio,ocf_to_debt,operating_cash_flow,net_profit,total_revenue,total_assets,total_liabilities,total_equity,cash_and_equivalents,interest_bearing_debt,interest_expense,current_liabilities,current_assets,inventory,accounts_receivable,other_receivables,prepayment,goodwill,construction_in_progress,operating_cost,operating_profit,capital_expenditure,investing_cash_flow,financing_cash_flow,interest_income,finance_cost,revenue_yoy,net_profit_yoy \
  --calculators inventory_turnover,receivable_turnover,npcf_ratio,asset_turnover,free_cash_flow,interest_coverage,net_debt_ratio,debt_to_ebitda,cagr,volatility \
  --years 5
```

### 港股注意事项
- 部分财务指标字段只有一年数据，需标注 🟡
- `parent_net_profit`、`total_revenue`、`operating_cash_flow`、`basic_eps` 等多年数据正常
- 无「营业成本」时用「销售成本」别名：`operating_cost = 营业额 - 毛利`

### 数据缺失处理

| 缺失项 | 影响分析项 | 补救措施 | 风险等级 |
|--------|-----------|---------|:--------:|
| 营收/利润数据 | 长期价值分析 | 标注🟡，降级评分 | 🟡 |
| 资产负债数据 | 偿债能力分析 | 标注🔴，终止分析 | 🔴 |
| 现金流数据 | 造假识别、偿债分析 | 标注🔴，终止分析 | 🔴 |
| 债务结构数据 | 偿债能力分析 | 标注⚠️，定性补充 | ⚠️ |

标注格式：
```
缺失字段：XXX
影响分析段：第X段（YYY）
补救措施：ZZZ
风险等级：🟡/⚠️/🔴
```

## 第三段：组合信号排查（排雷）

### 3.1 C01-C10 组合信号

| 编号 | 组合信号 | 计算公式 | 触发阈值 |
|:----:|---------|---------|---------|
| C01 | 净利润>0 + 经营现金流持续为负 | 累计经营现金流/累计净利润 | **<0.7（10年累计）** |
| C02 | 存货增长 >> 营收增长 + 毛利率上升 | 存货增速 - 营收增速 | >20% + 毛利率上升>5% |
| C03 | 应收增长 >> 营收增长 + 账龄缩短 | 应收增速 - 营收增速 | >30% + 1年内账龄占比上升>10% |
| C04 | 货币资金高 + 有息负债高 + 利息收入低 | 利息收入/货币资金 | 现金/负债>1 + 利息收入率<1% |
| C05 | 预付款暴增 + 长期挂账 | 1年以上预付款/总预付款 | 增长>50% + 1年以上>30% |
| C06 | 在建工程停滞 + 资金持续流出 | 进度%、累计投入/预算 | 进度<50%持续2年 + 投入>预算50% |
| C07 | 其他应收款暴增 + 与经营规模不匹配 | 其他应收/总资产 | >10% + 增长>营收增速50% |
| C08 | 关联交易占比高 + 毛利率异常 | 关联交易/营收 | >30% + 毛利率>同行20% |
| C09 | 商誉高 + 业绩承诺刚达标 | 商誉/净资产 | >30% + 完成率95%-105% |
| C10 | 折旧政策变更 + 利润刚好达标 | 折旧变更影响利润 | 年限延长 + 利润刚好达标 |

**C01 特别说明：** 多年累计值（≥5年）更能反映真实经营状况，单一年份易被操纵。累计<0.7表示现金流长期弱于利润，造假风险高。

### 3.2 判定标准

| 判定 | 符号 | 说明 |
|------|:----:|------|
| 未触发（安全） | ✅ | 无风险信号 |
| 部分触发 | ⚠️ | 需进一步核实 |
| 已触发（危险） | 🔴 | 高风险信号 |
| 数据不足 | 🟡 | 无法判定 |

### 3.3 输出格式

```
## 组合信号排查结果

| 编号 | 信号描述 | 累计值 | 判定 | 说明 |
|:----:|---------|-------|:----:|------|
| C01 | 累计经营现金流/累计净利润(10年) | **1.08** | ✅ | 累计>0.7 |
| C02 | 存货增速 - 营收增速 | 25%/22%/18% | ⚠️ | 部分年份触发 |
| ... | ... | ... | ... | ... |

风险汇总：
- 🔴 触发：X 项
- ⚠️ 部分触发：X 项
- ✅ 未触发：X 项

综合判定：🔴/⚠️/✅
```

## 第五段：长期价值分析（排优）

### 5.1 数据年限要求

| 要求 | 标准 | 少于标准的处理 |
|------|------|---------------|
| 最低年限 | 10 年 | 🔴 警告：无法评估长期价值 |
| 可接受下限 | 5 年 | ⚠️ 警示：仅能做初步评估 |
| 理想年限 | 15 年+ | ✅ 可穿越2-3个商业周期 |

### 5.2 长期增长分析（权重 25%）

| 指标 | 计算公式 | 优秀 (≥80) | 良好 (60-79) | 一般 (<60) |
|------|---------|:----------:|:------------:|:----------:|
| 营收 CAGR(10年) | (终值/初值)^(1/n)-1 | >15% | 8-15% | <8% |
| 净利润 CAGR(10年) | (终值/初值)^(1/n)-1 | >15% | 8-15% | <8% |
| 增长一致性 | 正增长年数/总年数 | >80% | 60-80% | <60% |
| 增长可预测性 | \|实际-预期\| | <10% | 10-20% | >20% |

### 5.3 稳定性分析（权重 35%）

| 指标 | 计算公式 | 优秀 (≥80) | 良好 (60-79) | 一般 (<60) |
|------|---------|:----------:|:------------:|:----------:|
| ROE 波动率 | 标准差/均值 | <20% | 20-40% | >40% |
| 毛利率波动率 | 标准差 | <5% | 5-10% | >10% |
| 现金流/净利润波动率 | 比值标准差/均值 | <20% | 20-40% | >40% |
| 资本开支/营收稳定性 | 占比标准差 | <3% | 3-5% | >5% |

### 5.4 可理解性分析（权重 20%）

| 指标 | 计算公式 | 优秀 (≥80) | 良好 (60-79) | 一般 (<60) |
|------|---------|:----------:|:------------:|:----------:|
| 主营业务占比 | 主营/总收入 | >90% | 70-90% | <70% |
| 收入集中度 | 前五大/总收入 | <30% | 30-50% | >50% |
| 会计政策稳定性 | 5年变更次数 | 0次 | 1次 | >2次 |
| 商业模式一致性 | 5年是否重大变化 | 无变化 | 小幅调整 | 重大变化 |

### 5.5 穿越周期能力（权重 20%）

| 指标 | 计算公式 | 优秀 (≥80) | 良好 (60-79) | 一般 (<60) |
|------|---------|:----------:|:------------:|:----------:|
| 2008危机表现 | 2008-2009 CAGR | 正增长 | 持平 | 负增长 |
| 2015放缓表现 | 2015-2016 CAGR | 正增长 | 持平 | 负增长 |
| 2020疫情表现 | 2020-2021 CAGR | 正增长 | 持平 | 负增长 |
| 危机后恢复速度 | 恢复至危机前年数 | <1年 | 1-2年 | >2年 |

### 5.6 长期价值评分卡

| 维度 | 权重 | 得分 (0-100) | 加权得分 |
|------|:----:|:----------:|:-------:|
| 长期增长 | 25% | - | - |
| 稳定性 | 35% | - | - |
| 可理解性 | 20% | - | - |
| 穿越周期能力 | 20% | - | - |
| **总分** | **100%** | - | **0-100** |

**评分标准：**

| 总分 | 判定 | 投资建议 |
|:----:|:----:|---------|
| ≥80 分 | 🟢 优秀 | 适合长期持有 |
| 60-79 分 | 🟡 良好 | 可关注，但有瑕疵 |
| <60 分 | 🔴 一般 | 不适合长期持有 |

## 第六段：时间序列分析

### 6.1 趋势对比

| 指标 | 计算公式 | 判定 |
|------|---------|:----:|
| 营收趋势 | CAGR=(终值/初值)^(1/n)-1 | ↑/→/↓ |
| 利润趋势 | 同比增速 | ↑/→/↓ |
| 利润率趋势 | 毛利率、净利率 | ↑/→/↓ |
| 现金流趋势 | 现金流/净利润 | ↑/→/↓ |

### 6.2 连续性异常检查

| 异常模式 | 计算公式 | 连续年数判定 |
|---------|---------|-------------|
| 经营现金流<净利润 | 现金流/净利润<1 | 2年⚠️ 3年🔴 |
| 存货增长>营收增长 | 存货增速-营收增速>20% | 2年⚠️ 3年🔴 |
| 应收增长>营收增长 | 应收增速-营收增速>30% | 2年⚠️ 3年🔴 |
| 毛利率>同行20% | 公司毛利率-同行>20% | 3年需解释 |
| ROE<10% | 净利润/净资产 | 连续3年🔴 |

### 6.3 输出格式

```
## 时间序列分析

趋势（5年）：
| 指标 | 趋势 | 说明 |
|------|:----:|------|
| 营收 | ↑ | CAGR XX% |
| 净利润 | → | 持平 |
| 毛利率 | ↓ | 下降X% |
| 现金流/净利润 | → | 约1.0 |

连续异常：
| 模式 | 年数 | 判定 |
|------|:----:|:----:|
| 现金流<净利润 | 3 | 🔴 |
| 应收>营收增长 | 1 | ✅ |
| ROE<10% | 0 | ✅ |
```

## 快速排雷模式（附录D）

> **适用对象**：时间有限的防御型投资者
> **预计用时**：30-60 分钟/家公司
> **规则**：8 项中任何一项🔴 = 直接排除，无需继续分析

| 序号 | 检查项 | 计算公式 | 风险阈值 |
|:----:|--------|---------|---------|
| 1 | 审计意见 | - | 非标准无保留🔴 |
| 2 | 现金流/净利润 | 经营现金流/净利润 | 连续2年<0.5🔴 |
| 3 | 资产负债率 | 总负债/总资产 | >70%🔴 |
| 4 | 利息保障倍数 | 经营利润/利息支出 | <3🔴 |
| 5 | CFO更换频率 | 5年CFO更换次数 | >3次🔴 |
| 6 | 关联交易占比 | 关联交易/营收 | >30%🔴 |
| 7 | 主营业务变更 | 重大变更次数 | >2次🔴 |
| 8 | 长期价值评分 | 10年数据评分 | <60分🔴 |

**判定流程：**
```
第 1 项🔴 → 直接排除（审计问题）
第 2-4 项🔴 → 直接排除（财务风险）
第 5-7 项🔴 → 直接排除（管理层/业务风险）
第 8 项🔴 → 不适合长期持有（长期价值不足）
```

**通过率参考**：A 股上市公司约 60-70% 能通过 8 项检查。

## 重要声明

1. **本手册为排雷工具，不是选股工具**
2. **阈值是参考，不是绝对** — 具体公司要结合行业特性判断
3. **组合信号优先级高于单字段** — 发现组合信号触发，应优先深入调查
4. **时间序列分析至关重要** — 单年异常可能是偶然，连续多年异常大概率有问题
5. **发现红色警报应停止投资** — 宁可错过，不要踩雷
6. **长期价值分析需要 10 年+ 数据**
7. **本手册基于历史数据** — 不预测股价走势，不预测公司未来业绩
8. **本手册依赖财报数据** — 不依赖外部研报和数据源
